This project creates a Graph Rag on Medi-Cal, IHSS documents, and hundreds of text files scraped from the Knowledge Base. Each of these docs and text files are nodes in the graph rag.

Knowledge Graph Building: Triples are used to build a directed graph . Triples are (Subject, Predicate, Object) e.g., (Insulin, treats, Diabetes). Nodes = entities, edges = relationships.

Hybrid Retrieval: A user question triggers: Graph-based retrieval: Find relevant triples (structured knowledge). Vector-based retrieval: Find semantically similar chunks. Both are merged into a context string. Both meaning: Context (Triples + Text Snippets(vectors)) and Question. Both of these are fed into LLM to get answer.

Pre-Process all the text files generated by 1. and remove the table of contents and other unnecessary texts running pp_text.py
